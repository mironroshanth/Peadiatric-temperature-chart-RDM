From the developer of this website



Hi, I'm Miron. I'm from SriLanka. I was in grade eight when I created this website. I obtained a little knowledge about programming when I was in grade seven through my school education. But I didn't have any practical knowledge of programming till my father showed me how to code using Notepad. After that I devoloped my knowledge on coding by creating simple html files through notepad and running them. Then in the same year I got to know about a software called Scratch.When my father saw me doing many projects through Scratch, he asked me whether I could create a project through which we can mark body temperatures of a sick person(because my father is a doctor he wanted a software to mark body
temperatures). After some days I finally created a temperature chart using Scratch. After my simple success I searched about Youtube to see how to code charts. I came across Google charts, Chart.js and many more websites till I found Highcharts. Using Highcharts GPT I obtained a good knowledge of coding charts. I also learnt from Chat GPT. After many days of coding my software I searched on the internet how to publish our code for free. With the help of Github I finally accomplished it. My website is still being continually updated by me.
