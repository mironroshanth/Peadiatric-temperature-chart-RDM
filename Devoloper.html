From the developer of this website

Hi, I'm Miron. I'm from Sri Lanka. I was in grade eight when I created this website. I obtained a little knowledge about programming when I was in grade seven through my school education. But I didn't have any practical knowledge of programming until my father showed me how to code using Notepad. After that, I developed my knowledge of coding by creating simple HTML files through Notepad and running them. Then in the same year, I got to know about a software called Scratch. When my father saw me doing many projects through Scratch, he asked me whether I could create a project through which we can mark the body temperatures of a sick person (because my father is a doctor, he wanted software to mark body temperatures). After some days, I finally created a temperature chart using Scratch. After my simple success, I searched on YouTube to see how to code charts. I came across Google Charts, Chart.js, and many more websites until I found Highcharts. Using Highcharts and GPT, I obtained a good knowledge of coding charts. I also learned from ChatGPT. After many days of coding my software, I searched the internet for how to publish our code for free. With the help of GitHub, I finally accomplished it. My website is still being continually updated by me.
