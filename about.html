About this website



This website is designed to mark body temperatures of children. But it can be also used to mark temperatures of adults if needed.

Before using this app,

  • Note that this app is made only for medical purposes.
  • Always remember fahreneit is the unit used in this website to mark the temperatures on the chart.
    • If the temperature was recorded in celsius you can convert it to fahrenheit using this website itself.
  • If you don't know how to mark temperatures properly read Getting started


The charts in theis website are from High Charts - https://www.highcharts.com
