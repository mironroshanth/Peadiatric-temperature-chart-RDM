About this website



This is a website used for marking the temperatures of a person. All the copy rights of this website belong to Daniel Miron Roshhanth. The website is created for medical purpose. If you are new to this website you can find the user instructios by clicking here or by reading the Getting started interface.

You can input the temperature recorded by your thermometer. Each time you input themperature the website wil automatically mark down the time the data was given.

Note that,
  • The temperature you input is in °F. If it s in °C you can convert it to °F by going to the temperature converter in the menu bar.
  • In this version the time will be marked in the Indian Standered Time.
