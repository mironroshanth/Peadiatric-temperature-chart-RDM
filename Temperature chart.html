Temperature Chart

This open-source website can be used to mark down body temperatures of a person. The temperature chart website is used for medical purposes.

For more details about the usage methods of this website, go through the getting started interface. To know more about this website, read the about this website page.

  • About this website

    • This is a website used for marking the temperatures of a person. All the copyrights of this website belong to Daniel Miron Roshhanth. The website is created for medical purposes. If you are new to this website you can find the user instructions by clicking here or by reading the Getting started interface.

    • You can input the temperature recorded by your thermometer. Each time you input the temperature, the website will automatically mark down the time the data was given.
    • The temperature you input is in °F. If it is in °C, you can convert it to °F by going to the temperature converter in the menu bar.
    • In this version, the time will be marked in Indian Standard Time.



  • Getting Started

    • This website is used to mark the body temperatures of a person. The methods of using the app are explained below.


    First, click the new chart button on the menu bar to access the page as shown above. Then enter the name and date of birth (MM-DD-YYYY) of the person as shown below.


    Then input the temperature you measured. Make sure the temperature was measured at the time you input the value. Also note that the temperature you enter is in Fahrenheit. If it is in Celsius, click here to convert to Fahrenheit.


    Click the 'Add data' button to mark the temperature as shown below.


    Click the 'Save' button to save the file.
